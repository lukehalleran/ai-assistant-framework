Benchmark infrastructure:
- New scripts/benchmark_retrieval.py — standalone runner producing Recall@1/3/10,
  MRR (aggregate + per-intent), rank distribution histogram, prompt build
  latency (p50/p90/p95), cold vs warm cache timing. Appends to
  data/benchmark_history.json for tracking over time.
- docs/BENCHMARK_METRICS.md — metrics tracking doc with current values,
  per-case detail, historical tables.
- BenchmarkResult extended with recall_at_1, recall_at_3, rank_positions,
  has_retrieval_requirement. Aggregation now excludes 6 intent-only cases
  (casual/meta/general) from recall/precision/MRR averages — previously
  inflated by trivial 1.0 contributions.

Temporal decay fix (memory_scorer.py):
- rank_memories() now uses time_manager.current() instead of datetime.now()
  for age calculation. Fixes benchmark where MockTimeManager reference time
  diverged from wall clock, making all seed memories appear ~2100h old and
  rendering temporal anchors useless.
- Two-regime temporal decay: small anchors (<=48h, "today") use flat plateau
  within window (recency 1.0→0.85), steep dropoff outside. Large anchors
  (>48h, "last week") peak at anchor age (floor scales with anchor size),
  penalizing too-recent memories. Temporal recall MRR: 0.115 → 1.000.
- MockTimeManager.current() added in conftest.py and benchmark script.

Scoring tuning (memory_scorer.py + intent_classifier.py):
- _stem() suffix stripper for token overlap matching — handles
  anxious/anxiety, deployed/deployment, features/feature without full
  Porter stemming (too aggressive for code symbols like asyncio).
- Tag-keyword bonus: stemmed query tokens matching stemmed memory tags
  add up to +0.15 to continuity score.
- Weight rebalancing across 5 intents — shifted from recency to continuity:
  FACTUAL_RECALL (rec 0.10→0.05, cont 0.05→0.10),
  EMOTIONAL_SUPPORT (rec 0.30→0.15, cont 0.25→0.40),
  TECHNICAL_HELP (rel 0.45→0.40, rec 0.15→0.10, tru 0.20→0.15, cont 0.05→0.20),
  CREATIVE_EXPLORATION (rec 0.15→0.10, cont 0.15→0.20),
  PROJECT_WORK (rec 0.15→0.10, tru 0.25→0.20, cont 0.05→0.15).
- Result: R@3 0.69→1.00, MRR 0.66→0.79, all 14 expected items now in top 3.
