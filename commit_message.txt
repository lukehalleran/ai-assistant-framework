feat: standalone memory scorer — 12-step ranking + evidence-based truth

Single-file memory/document scoring engine extracted from Daemon, zero external
dependencies. Drop-in ready for any Python 3.9+ RAG system.

memory_scorer.py (680 lines, stdlib only)
- MemoryScorer: 12-step ranking pipeline — relevance + collection boost,
  two-regime temporal decay (small/large anchor), evidence-based truth,
  importance, stemmed token continuity + tag matching, structural alignment
  (math-aware), topic match, deictic follow-up handling, meta-conversational
  bonus, optional graph proximity boost, staleness penalty with steep threshold
- TruthScorer: stateless evidence-based truth — source-dependent initial scores,
  confirmation boost, correction/contradiction penalties, weekly time decay
  toward configurable floor. All state lives in document metadata.
- ScorerConfig: single dataclass with all tunable knobs and sensible defaults
- GraphScorerProtocol: optional runtime-checkable protocol for plugging in
  any knowledge graph backend
- CLI: `python memory_scorer.py demo|score|truth`

Key design choices vs Daemon's version:
- TruthScorer is instance-based (config-injectable) instead of static methods
- Graph integration via clean Protocol instead of importing graph_utils
- time_fn parameter replaces time_manager dependency
- All config constants inlined as ScorerConfig defaults matching production values
