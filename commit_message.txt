## Obsidian Vault Integration

Adds personal notes from user's Obsidian vault (~Documents/Luke Notes) to the
context retrieval system. Notes are embedded in a dedicated ChromaDB collection
and retrieved using hybrid search (1/3 keyword + 2/3 semantic).

**New file:** `knowledge/obsidian_manager.py`
- Smart chunking: whole notes <1500 chars, else split by ## headers
- Extracts #tags and [[wiki links]] as metadata
- Hybrid retrieval: keyword match on title/tags + semantic similarity

**CLI commands:**
- `python main.py embed-vault` - Index vault
- `python main.py embed-vault --force` - Force re-index
- `python main.py vault-stats` / `clear-vault`

## News-Aware Web Search

Tavily searches now auto-detect news queries and use the optimized news agent.

**Changes:** `knowledge/web_search_manager.py`
- Added `_is_news_query()` detection (keywords: today, happened, news, latest)
- Passes `topic="news", days=1` to Tavily for current events queries
- Logs: "[WebSearch] News query detected, using topic='news', days=1"

## Fact Correction Detection

User corrections now extract with HIGH confidence (0.92) to supersede stale facts.

**Changes:** `memory/fact_extractor.py`
- Added `_detect_corrections()` method
- Patterns: "I'm X, not Y", "actually I'm X", "not Y, I'm X"
- 0.92 confidence triggers UserProfile's supersedes logic
- Corrections run first in extraction pipeline
